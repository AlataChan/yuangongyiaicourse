  1. 单选 手册·助理架构首页·架构

    项目助理 AI智能体的核心构成，不包括下列哪一项？

    • A统一入口
    • B多技能路由
    • C知识库增强
    • D替换掉全部人工
    显示答案与解析
    答案：D

    项目助理是「统一入口 + 多技能路由 + 知识库增强」，是辅助而非替代人。

  2. 单选 手册·意图路由首页·架构

    AI智能体收到用户输入后，通常的第一步是？

    • A意图识别（文书 / 案例 / 传播 / 综合问答）
    • B直接生成结项报告
    • C删除知识库
    • D输出缺失项清单
    显示答案与解析
    答案：A

    先识别意图，再路由到对应技能与知识库。

  3. 多选 手册·Agent平台 首配首页·配置步骤

    在 Agent平台 首配项目助理时，通常需要修改哪些？（多选）

    • A机构名
    • B欢迎语
    • C系统提示词 tone
    • D别人的账号密码
    显示答案与解析
    答案：A、B、C

    Fork 母版后改机构名、欢迎语、tone；不涉及他人账号密码。

  4. 判断 手册·连接资产首页·配置步骤

    知识库 RAG 的作用，是让智能体回答时能引用机构自己的事实与案例。

    • B错

    答案：对

    RAG 让回答基于机构知识库，而非泛泛而谈或编造。

  5. 单选 手册·工作流管道首页·工作流

    M1 文书链工作流的典型顺序是？

    • A输入 → 信息抽取 → 知识库检索 → 生成 → 缺失项清单
    • B输入 → 直接生成
    • C检索 → 删除 → 输出
    • D随机顺序
    显示答案与解析
    答案：A

    抽取素材→检索历史项目书/资助方偏好→生成→提示缺失项。

  6. 单选 手册·工作流管道首页·工作流

    M2 案例库工作流中，写入知识库前必经的节点是？

    • A脱敏规则节点
    • B付费节点
    • C联系资助方
    • D打印存档
    显示答案与解析
    答案：A

    原始记录先经脱敏，再结构化、写入知识库或输出案例卡片。

  7. 多选 手册·互测首页·互测

    第 3 课互测收集的三类反馈包括？（多选）

    • A准
    • B不准
    • C乱答
    • D界面好看
    显示答案与解析
    答案：A、B、C

    扫码互测收集「准 / 不准 / 乱答」三类，用于定位改进点。

  8. 判断 手册·系统提示词首页·配置步骤

    在系统提示词里嵌入各技能的触发条件，是为了让智能体「该用哪个技能才用哪个」。

    • A对
    • B错
    显示答案与解析
    答案：对

    触发条件 = 路由依据，避免误调用或漏调用技能。

  9. 单选 手册·分层作业首页·互测

    第 3 课 L1「用起来」层的最低验收要求是？

    • A对话 Bot 跑通 + 自己 10 问

    答案：A

    L1：对话 Bot + 1 知识库，自测 10 问；L2 升级为多步工作流 + 插件 + 内测。

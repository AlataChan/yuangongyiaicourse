如何使用这些技能

1
在课程平台新建 Bot（Agent）

进入 Agent 平台，按第 1 课讲义「新建 Bot → 配置 System Prompt」流程操作。

2
复制对应 SKILL.md 中的 System Prompt

点击上方「查看技能」，在 SKILL.html 页面找到 System Prompt 区域，复制完整内容粘贴进 Bot 配置。

3
按触发词发起对话

每个技能都标注了推荐触发词（如「写结项报告」），直接发送触发词 + 附件即可启动工作流。

4
Fork 后按机构需求调整

如需修改输出格式或增加机构特有字段，参考第 1 课「技能定制」讲义，在 SKILL.md 基础上修改即可。技能体检（skillopt）可帮你验证改动质量。

元公益工作室

元公益（北京市公鸣社会工作发展中心）成立于2015年，专注于为公益组织提供战略管理与行业发展的咨询服务，助力其战略管理能力提升、组织发展和转型。同时，以咨询机构和技术支持角色，积极推动公益组织的数字化应用。截止目前，已为超过100家公益组织（包括基金会、民非组织、协会）提供了组织发展、战略咨询或数字化赋能等服务。

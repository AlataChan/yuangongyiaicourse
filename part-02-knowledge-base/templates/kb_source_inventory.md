> 用途：第 2 课课前或课堂使用，先把散乱资料清点出来，再决定哪些进个人知识库、工作台、腾讯元器或高阶 RAG 工具。

## 使用步骤

1. 每个机构先填 10-20 份最常用资料，不追求一次盘完整。
2. 逐份判断合规档位：L1 公开、L2 准公开、L3 内部、L4 敏感。
3. L4 直接剔除；L3 仅留个人知识库 / 工作台；L1/L2 才进入组织层腾讯元器。
4. 判断是否已经清洗：是否去噪、脱敏、结构化、补 frontmatter。
5. 对每份资料写 1 个“它应该回答的问题”，后续作为命中率测试题。

## 快速判断

| 情况 | 推荐动作 |
|------|----------|
| 公开发布过 | 标 L1，清洗后可进腾讯元器 |
| 不含敏感信息，但不是公开资料 | 标 L2，脱敏和负责人审核后再进腾讯元器 |
| 团队内部工作资料 | 标 L3，留在个人知识库 / 工作台，不进组织层知识库 |
| 包含受益人、志愿者、捐赠人个人信息 | 标 L4，不进入任何知识库 |
| 内容很有价值但格式很差 | 先用 OCR / LLM 清洗成 LLM-wiki 条目，再判断目标位置 |

# ima + WorkBuddy 知识工作流练习

> 文件路径：`/Users/apple/Documents/4.0 Sanyuan/2.4 环境公益"新"力量/course/part-02-knowledge-base/tools/ima_workbuddy_lab.md`
>
> 用途：第 2 课初阶练习。目标不是精通工具，而是把个人散乱资料变成可被任务调用的知识。

## 适用范围

| 适合放入 ima / WorkBuddy | 不适合放入 |
|--------------------------|------------|
| 个人常读 PDF、网页、公开报告、已脱敏案例 | L4 敏感信息 |
| 需要快速摘要、问答、萃取的微信文件 | 未经授权的内部原始资料 |
| L3 内部资料的个人辅助阅读 | 需要组织共享的正式知识库唯一版本 |

## 课堂任务

1. 选择 5 份资料：至少包含 1 份 PDF、1 份 Word 或 Markdown、1 份网页或图片资料。
2. 导入 ima，分别提问：
   - 这份资料讲了什么？
   - 里面有哪些可复用案例？
   - 哪些内容适合进入组织知识库？
   - 哪些内容需要脱敏？
3. 让 ima 输出候选标签：
   - 文档类型
   - 年份
   - 项目名
   - 领域标签
   - 合规档位建议
4. 在 WorkBuddy 中调用整理后的资料，跑一次第 1 课技能任务。
5. 记录一次任务结果：哪些事实被正确引用，哪些仍需要进腾讯元器。

## 课堂输出

```text
我用 ima 整理了 ___ 份资料；
其中 ___ 份适合进入腾讯元器；
___ 份只适合留在 WorkBuddy；
我用 WorkBuddy 完成了 ___ 任务；
需要补充或脱敏的资料是：___。
```

## 常见误区

| 误区 | 修正 |
|------|------|
| 把 ima 当正式组织知识库 | ima 更适合作个人资料收纳、阅读和初步萃取 |
| 资料一导入就上云 | 先做 L1-L4 分级，L2 脱敏审核后再上云 |
| 只问“总结一下” | 要追问“哪些事实可引用、哪些内容需脱敏、适合打什么标签” |
| 只看回答是否顺畅 | 还要检查是否能指出来源和边界 |

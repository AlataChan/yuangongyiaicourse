## 助理画像

| 维度 | 内容 |
|------|------|
| 一句话 | 户外教育线路/招募/社群运营/推文短视频一条龙（从 0 建库） |
| 主模块 | M3 |
| 输入 | 领队 SOP 草案、线路方案、活动反馈 |
| 输出 | 可执行 SOP、招募模板、公众号稿、短视频脚本、社群话术 |
| 成功标准 | 每周 1 篇公众号 + 1 条视频脚本；关注每周 +5–10 |

## 最简 MVP（第 3 课结束前）

**1 条线路 + 1 次招募**：输出招募推文初稿 + 社群 FAQ 10 条 + 领队 SOP 一页纸。

| Skill | 用途 |
|-------|------|

## QoderWork 三步

1. 建 `户外教育/`：`SOP/`、`线路/`、`招募/`、`复盘/` 空文件夹，先填 1 条线路样本。
2. 知识库从 0 开始：第 2 课至少入库 SOP 模板 + 1 次历史活动记录。
3. 元器 Bot 挂 FAQ + recap，招募走单独触发词。

## 分课卡点

| 课次 | 做什么 |
|------|--------|
| 第 1 课 | SOP Skill 或 recap 二选一先跑通 |
| 第 2 课 | 最小知识库（≥5 份） |
| 第 4 课 | 补齐第二 Skill + 路由（SOP vs 推文 vs FAQ） |
| 第 5 课 | 演示完整招募闭环 |

## 常见坑

- 知识库为 0 仍求「自动发布」→ 先 MVP 文案，发布人工确认。
- 「自动运行招募」→ 课程交付为模板+脚本，支付/保险接口不在课内。

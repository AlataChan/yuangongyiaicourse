# SG-09 · 多站点数据台账型 · Skill 构建指引（Lite）

> 对应需求代号 T09 | M2 · 高原/野外多站点项目 | 与 [SG-12](./SG-12-草原多站点台账型.md) 互参

---

## 助理画像

| 维度 | 内容 |
|------|------|
| 一句话 | 多站点日报 CSV + 照片说明 → 周报/月报初稿，人工核对后定稿 |
| 主模块 | M2 |
| 输入 | 各站点每日台账表、工作照文字说明 |
| 输出 | 周期周报/月报、数据核对表、可选视频脚本（第二 Skill） |
| 成功标准 | 周报撰写 2h → ≤0.5h；缺失站点/日期能自动标出 |

## 最简 MVP（第 2 课结束前）

**1 个项目 × 2 个站点 × 7 天样本** 收进 QoderWork 统一文件夹 → 跑通 `env-ngo-field-ledger` → 产出 1 份可校订周报草稿。

```text
01_原始台账/站点A、B/  →  Skill  →  02_周期汇总/2026-Wxx_周报草稿.md
```

## 推荐 Skill

| 顺序 | Skill | 用途 |
|------|-------|------|
| 1 | [`env-ngo-field-ledger`](../part-04-scenario/skills/env-ngo-field-ledger/SKILL.md) | 台账汇总 → 周报/月报 |
| 2 | [`env-ngo-activity-recap`](../part-04-scenario/skills/env-ngo-activity-recap/SKILL.md) | 同周期素材 → 公众号推文 |
| 3（可选） | 视频脚本 Skill（content-pipeline 延伸阅读） | 分镜/口播稿，不做剪辑 |

## QoderWork 三步

1. 按 [文件夹结构](../part-04-scenario/skills/env-ngo-field-ledger/assets/folder_structure.md) 建 `<项目>_台账/`。
2. 收拢 1 周 CSV + 照片说明，统一命名。
3. 加载 field-ledger Skill，指定周期，输出到 `02_周期汇总/`。

## 分课卡点

| 课次 | 做什么 |
|------|--------|
| 第 1 课 | 写清 field-ledger 触发说明 + 输出契约（Fork 样板） |
| 第 2 课 | 文件夹归集 + 知识库挂载台账目录 |
| 第 4 课 | 第二 Skill：activity-recap；路由：「写周报」vs「写推文」 |
| 第 5 课 | 演示：1 份核对后的周报 + 1 篇推文 |

## 常见坑

- 未归集就调用 Skill → 先完成文件夹，不跨电脑找文件。
- 不同单位混加（袋/公斤）→ 分表汇总。
- 缺日报不标注 → 必须进「数据核对说明」。

## 助理画像

| 维度 | 内容 |
|------|------|
| 一句话 | 草原/边境多站点每日台账 → 周报/月报 + 传播脚本 |
| 主模块 | M2 |
| 输入 | 捡拾分类、巡护、野生动物登记等 CSV；照片说明 |
| 输出 | 周期报告初稿、推文/短视频脚本 |
| 成功标准 | 同 SG-09：周报 2h → ≤0.5h |

## 最简 MVP

与 SG-09 相同：**2 站 × 7 天** 归集 → `env-ngo-field-ledger` → 1 份周报草稿。

## 推荐 Skill

| Skill | 用途 |
|-------|------|
| [`env-ngo-field-ledger`](../part-04-scenario/skills/env-ngo-field-ledger/SKILL.md) | 台账 → 报告（主 Skill） |
| [`env-ngo-activity-recap`](../part-04-scenario/skills/env-ngo-activity-recap/SKILL.md) | 报告定稿后 → 公众号稿 |
| 视频脚本（content-pipeline 延伸） | 口播/分镜，不剪辑 |

## QoderWork 三步

1. 共用 [folder_structure](../part-04-scenario/skills/env-ngo-field-ledger/assets/folder_structure.md)。
2. 照片说明格式固定：`谁在做什么 + 拍摄者` 单行一条。
3. 定稿报告放 `03_已定稿/`，供第 2 课知识库。

# SG-16 · 固废科普项目管理型 · Skill 构建指引（Lite）

> 对应需求代号 T16 | M3/M2 · 与 [SG-14](./SG-14-湿地项目管理型.md) 同型 + 法规资讯

---

## 助理画像

| 维度 | 内容 |
|------|------|
| 一句话 | OKR/进度清单 + 案例报告 + 固废科普知识库 + 法规热点资讯摘要 |
| 主模块 | M3（作业自评）/ 能力同 M2 项目管理 |
| 输入 | 项目书、节点、案例记录、法规/热点链接 |
| 输出 | OKR 表、案例报告、资讯摘要、科普文案 |
| 成功标准 | 同 SG-14 + 每周 1 份资讯摘要 |

## 最简 MVP

**复用 SG-14 MVP** + **1 份「本周固废/环保资讯摘要」**（3–5 条，每条 2 句+来源链接）。

## 推荐 Skill

| Skill | 用途 |
|-------|------|
| SG-14 全套（OKR + case-extract + FAQ） | 项目管理主链 |
| 自建「资讯摘要」Skill | 输入 URL/标题列表 → 输出摘要+行动建议 |
| content-pipeline · content-research-writer-cn | 深度科普稿（延伸阅读） |

## QoderWork 三步

1. 与 SG-14 共用 `项目管理/` 结构。
2. 增 `资讯/` 文件夹：存每周候选链接 Markdown。
3. 资讯 Skill 与 OKR Skill 分触发词：「摘要资讯」vs「生成 OKR」。

## 分课卡点

| 课次 | 做什么 |
|------|--------|
| 第 1–3 课 | 对齐 SG-14 |
| 第 4 课 | 资讯摘要为第二或第三 Skill |
| 第 5 课 | OKR 表 + 资讯摘要 + 1 篇科普稿 |

## 常见坑

- 与 SG-14 学员可结对：共用 OKR/案例 Skill，互测路由。
- 资讯 Skill 不编造政策 → 无全文则只输出「待阅读原文」。

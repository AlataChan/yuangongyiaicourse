📚 技能样板库

一页看全四个技能，
点开即读，无需打开源码目录

项目文书链的三个即用技能样板，外加一个「技能优化」工具样板，都在这里。点击「预览技能」即可在线阅读完整的技能定义与配套模板， 排版清晰、可一键复制——不用下载文件、也不用面对一堆 Markdown 源码。挑一个改成你机构的版本， 再到第 3 课汇入你的「项目助理」AI智能体。

场景 · 结项报告

结项报告技能

把项目执行流水账转成结构稳定、可直接交给资助方的结项报告。

触发：结项 / 项目总结 / 执行流水账 / 向资助方汇报

执行流水账 → 六章节结构化报告

场景 · 项目书 / 提案

项目书技能

用机构简介、过往案例与资助方指南，组装一份项目立项书初稿。

触发：项目书 / 立项书 / 项目申请书 / 申报方案

机构简介 + 案例 + 指南 → 立项书初稿

查看页面
场景 · 资助申请信

资助信技能

把项目核心信息浓缩成一封简短、有说服力的资助申请信或项目摘要。

触发：资助申请 / 申请信 / 意向书 / 一页纸摘要

项目核心信息 → 一页纸申请信 / 摘要

查看页面
工具 · 技能优化（skillopt）

技能优化技能

不写文书——给你写好的技能做「体检」：10 项清单打健康分、生成评测用例、给最小修改建议。

触发：优化技能 / 不触发 / 误触发 / 评测没过 / 改 description

你的 SKILL.md → 体检报告（健康分 + 可粘贴改写文本）

查看页面

想直接上手？预览面板里点「复制技能定义」把 SKILL.md 粘进 WorkBuddy 的角色设定，再「复制模板」上传配套模板即可——课堂上一个角色就是一个最小可用的技能，第 3 课再迁到腾讯元器。

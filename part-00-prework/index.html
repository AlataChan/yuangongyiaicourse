第一步 · 概念准备

先读懂项目助理 Agent 的工作方式

这份速读不是技术百科，而是第 1–3 课的共同语言：LLM 怎么用上下文工作，为什么上下文越满越容易混淆，知识库为什么要切片、检索和重排，以及 Agent 为什么必须有权限和人工审核边界。

读完要理解的 4 组关系

  • LLM 会生成，不等于会查证；项目事实必须来自可靠资料
  • 256k / 1M 是总上下文窗口，不代表越塞越稳定
  • embedding 按语义找资料，切片通常不建议超过 3000 中文字
  • Skill 管「怎么做」，知识库管「依据什么」，Agent 管「如何编排」

阅读 AI 基础概念速读

带着 3 个答案去填立项书

  • 我的项目助理 Agent 主要服务对象是谁？
  • 它最需要查找或引用哪三类资料？
  • 它不应该接触或输出哪三类信息？
  • 如果做成 Skill，最需要写清哪三条规则？

第二步 · 环境准备

开通 LLM 模型 API

第 1 课在 QoderWork 上调试技能时需要连接大模型 API。组织层 Agent平台 Bot 不需要自购 Token——以下准备面向个人层与可选自建路线。

主线 · 阿里云百炼

  • 注册阿里云并完成实名认证
  • 在百炼控制台开通大模型推理服务
  • 选择华北 2（北京）地域，创建 API Key
  • 新用户通常有 90 天免费额度；用尽后按量充值
  • 建议机构预留 50–100 元余额用于学习期调试

打开百炼 API Key 页

辅助 · DeepSeek 开发者平台

  • 注册并完成实名认证
  • 在 API keys 页创建 Key（仅显示一次，立即保存）
  • 充值页面用支付宝 / 微信充值（建议首次 50 元）
  • 可作为百炼额度用尽时的备选模型来源
  • QoderWork 中 Base URL：https://api.deepseek.com

打开 DeepSeek 开放平台

安全提醒：API Key 是计费凭证，禁止写入立项书、Git 或公开文档。仅由 1–2 位课程接口人保管；发现异常调用请立即在控制台删除 Key。

完整图文步骤、QoderWork 配置参数与自检清单见 《LLM API 开通指南》。

项目助理立项书
* 一句话目标
例：帮我把项目执行流水账变成可提交的结项报告
* 主场景模块
M1 项目文书链 M2 案例与知识沉淀 M3 传播与运营
* 输入
例：项目日志（Word）、活动照片清单、资助方模板
* 输出
例：六章节结项报告（背景 / 活动 / 产出 / 成效 / 反思 / 缺失项）
* 需要的技能（≥1）
例：结项报告技能、项目书技能
需要的知识库
例：历年项目书 + 资助方指南 + 机构模板
* 上线 2 周成功标准
例：结项报告从 3 天缩短到半天，且不遗漏章节
立项书中不要填写真实受益人、捐赠人、志愿者姓名与联系方式。如需提及，请用 A / B / C 代替。
场景选择

3 个可选场景模块

作业里选定一个主场景模块，差异只在技能组合与知识库侧重，最终都汇入同一个「项目助理AI智能体」。

📝

M1 项目文书链

默认主模块
  • 结项报告 / 项目书 / 资助信技能
  • 知识库：资助指南 + 历史项目书 + 机构模板
  • 目标：结项报告耗时下降一半以上
选择条件：文书写作是最高频痛点，或无法判断时默认选此
🗂️

M2 案例与知识沉淀

首要痛点
  • 案例萃取 / 服务故事脱敏 / 案例检索技能
  • 知识库：历年案例 + 标签体系 + 服务故事
  • 目标：案例检索命中率达 70% 以上
选择条件：资料散乱、经验无法复用是最大痛点
📣

M3 传播与运营

进阶可选
  • 活动复盘 / 公众号稿 / 社群 FAQ 技能
  • 知识库：历史推文 + 品牌话术 + 活动日历
  • 目标：单场活动复盘文案 20 分钟内完成
选择条件：传播产出是核心需求，建议同时挂 M1 或 M2 作为基础

说不清选哪个？默认 M1 项目文书链——所有机构都要写文书，覆盖面最广。第 4 课还有机会深化第二个模块。

提交与答疑

怎么交 · 哪里问

异步为主、答疑为辅。下面把截止、提交方式与可选答疑一次讲清。

提交方式

  • 下发时间：开课前 1 周随作业包发出
  • 截止时间：第 1 课开课前
  • 提交渠道：课程协作空间（在线表单 / 文档模板回填，具体链接随作业包下发）
  • 未按时提交不影响参加第 1 课，但需课后补交

可选集中答疑（60 min）

  • 开课前 1 周内安排 1 次线上集中答疑，自愿参加
  • 用途：解答立项书填写疑问、帮拿不准主场景模块的机构当场判断
  • 这不是 1v1 门诊，缺席不影响开课，疑问也可在课程社群异步提出

第 1 课开场会用 15–20 分钟做作业快览 + 共性问题点评，不要求每家现场逐一陈述。

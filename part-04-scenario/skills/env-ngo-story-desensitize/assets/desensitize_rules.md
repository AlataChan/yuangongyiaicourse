# 服务故事脱敏规则

> 适用范围：需公开发布的服务故事、受益人叙事、访谈记录、案例文本。

## 脱敏对照表

| 信息类型 | 原始示例 | 脱敏处理 | 输出示例 |
|----------|----------|----------|----------|
| **真实姓名** | 张三丰 | 替换为字母代号，同一人全文统一 | 受益人 A |
| **手机号** | 138-1234-5678 | 直接删除 | [已删除] |
| **身份证号** | 110105199001011234 | 直接删除 | [已删除] |
| **银行卡号** | 6222 0000 1234 5678 | 直接删除 | [已删除] |
| **精确金额** | 43,872 元 | 量级范围 | 约 4 万元 |
| **详细地址** | 北京市海淀区中关村南大街 5 号 | 模糊到区县 | 北京市海淀区某社区 |
| **机构全称** | XX 市绿源环保公益中心 | 机构简称或代号 | <机构简称> |
| **照片中人脸** | 清晰正面照描述 | 标注需打码 | [建议对人脸打码后发布] |
| **未成年人信息** | 小明，12 岁，XX 小学六年级 | 代号 + 删除学校 | 学生 B，12 岁 |
| **疾病/残障信息** | 张三确诊 XX 病 | 模糊描述 | 受益人 A 因健康原因 |

## 处理原则

1. **同一人统一代号**：首次出现分配 A/B/C，后续全文一致。
2. **删除优先于遮盖**：手机号、证件号不做 `138****5678` 式部分遮盖，直接删除。
3. **金额用量级**：千元级→「约 X 千元」，万元级→「约 X 万元」，百万级→「约 X 百万元」。
4. **不改变事实**：脱敏是保护隐私，不是美化或修改事件经过。
5. **存疑则标注**：不确定是否需脱敏的内容，标注 `[请确认是否需脱敏]` 交由用户决定。
6. **未成年人从严**：涉及未成年人的信息，学校名称、班级、家庭住址一律删除。
